1# ai-chatbot-web-app
AI chatbot web application built using Python and Flask that processes user input and generates responses using basic NLP techniques.

## Tech Stack
- Python
- Flask
- HTML
- CSS
- JavaScript

## Features
- Interactive chat interface
- Processes user queries
- Generates automated responses
- Beginner-friendly design

## Project Purpose
This project was created to learn chatbot development, backend logic, and basic AI concepts.
